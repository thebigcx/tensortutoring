# Welding

<table>
  <tr>
   <td><strong><em>Welding</em></strong>
   </td>
   <td><strong>Notes</strong>
   </td>
  </tr>
  <tr>
   <td>Resistance
   </td>
   <td>
<ul>

<li>Pressure + current causes melting together</li>

<li>Butt, seam, spot</li>
</ul>
   </td>
  </tr>
  <tr>
   <td>Oxy-Acetylene
   </td>
   <td>
<ul>

<li>Melted by oxy flame, filler metal added</li>

<li>Earliest welding</li>

<li>Can weld steel, cast iron, and some non-ferrous alloys</li>
</ul>
   </td>
  </tr>
  <tr>
   <td>Electric Arc
   </td>
   <td>
<ul>

<li>Melted by electrode which acts as filler (consumed)</li>

<li>Flux used to prevent oxidation</li>

<li>Faster than oxy</li>
</ul>
   </td>
  </tr>
  <tr>
   <td>MIG
   </td>
   <td>
<ul>

<li>Continuous (consumable) feed wire/electrode – faster</li>

<li>Inert gas (e.g. argon) to prevent oxidation</li>

<li>Easily automated - used extensively in manufacturing</li>
</ul>
   </td>
  </tr>
  <tr>
   <td>TIG
   </td>
   <td>
<ul>

<li>Tungsten (non-consumable) electrode</li>

<li>Filler rod</li>

<li>Inert gas shield</li>

<li>Stainless steel, <strong>aluminium & titanium alloys – </strong>welds reactive metals</li>
</ul>
   </td>
  </tr>
</table>


Microstructural Changes



* Small equiaxed grains in centre of filler
* Columnar grains growing outwards
* Recrystallisation in HAZ – large equiaxed grains
* Finer recrystallised grains at transition to the (cold-rolled) original grains
